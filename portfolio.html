Equilibrium Simulator

An interactive demonstration that models equilibrium in chemical reactions. The reaction is modeled as bouncing particles which can move between opposing sides of the reaction. From the control panel, the user can modify the concentration, temperature, and required energy to observe how each can affect the equilibrium of the reaction. This program can be used as an educational tool for teaching equilibrium reactions in chemistry.

Github Link : github.com/jpngfile/EquilibriumSimulator Tools used : Java

Maliable Pngball

An installable Java-based two-dimensional pinball game. It features a 2D physics engine that supports multiple object collision and acceleration. The game also has an educational component, which will present physics questions to the user and will grant additional lives if answered correctly. Overall, an excellent program for demonstrating and teaching physics in an fun manner. Co-created with Aakash Mali.

Github Link : github.com/jpngfile/Maliable-PngBall Tools used : Java, HTML, CSS

SUYOP (Summer Youth Opportunities)

A website that showcases extracurricular activities for high school students in the Toronto area. It features over 30 handpicked programs that vary in activity but are all of high quality so you are sure to find one that is to your liking. The programs can be filtered by category, location and time commitment, as well as be displayed in a table format for easy comparison.

Site Link : Suyop.com Tools used : HTML, CSS, javascript, jquery, bootstrap
